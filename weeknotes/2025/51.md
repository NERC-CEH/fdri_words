## üéÑScatterplot / Variable v Variable / Rsquared is LIVE!

This is now available. To compare two variables, check the corresponding box in the "compare" column in the table at the top of the explore tab. Once two variables are selected they will be able to be seen using the "compare" tab just above the main plot. The R Squared value is calculated automatically and the R Squared trendline is plotted in orange

## ‚òÉÔ∏è NRFA Batches page

A new page has been added for NRFA which lists all uploaded batches and their associated datasets. It can be found here: https://dri-ui.staging.eds.ceh.ac.uk/nrfa/batches
To view the datasets, click on the name of the batch and the view will expand to display the datasets. Clicking on "Plot Data" will open a new tab containing the explore plot with the full dataset plotted for the batch.
