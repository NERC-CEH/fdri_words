## Flux

The work on getting the pipeline deployed to AWS has continued, albeit somewhat interrupted by the NCDR Science Area Meeting.  We have unfortunately found that the telemetry system isn't as robust as we might have hoped: querying the logger's HTTP API often returns empty files with no data in, so it looks like we'll have to rework it to use the FTP interface instead.
