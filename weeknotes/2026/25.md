## Flux

After what looked like a promising start using the built-in [Campbell Web Service API](https://www.campbellsci.com.au/web-api-article) to collect flux data directly from field devices, we found it was frequently returning empty results but that was happening more consistently over time. So we've switched to collecting data hourly, via FTP.

The existing code in the ingester to read and transform the binary `tob3` format was really useful for this. IT Infrastructure quickly deployed the new version, and it's ready to add more flux sites as they go out into the field - Mill Farm next.
